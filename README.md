# Planning Poker

A real-time planning poker application for agile teams built with Next.js and WebSockets.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the WebSocket server (in one terminal):

```bash
npm run dev:ws
```

3. Start the Next.js development server (in another terminal):

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to use the app.
